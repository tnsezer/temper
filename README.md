Temper Assessment
===============

## Installation
```bash
cd public/ && php -S localhost:8080
```
OR
```bash
cd docker/ && docker-compose up -d --build
```

##### You can visit http://localhost:8080

## How to run tests?

```bash
vendor/bin/phpunit -c phpunit.xml
```
